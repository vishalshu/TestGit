== Description ==

Smart AD Tags is an easy to use all in one advertisement package plugin which helps bloggers to insert advertisement in various locations of a post. Besides that, you can also insert ad on the sidebar of your blog, comment area as well as on any individual page. Preview banners when editing them. This plugin will help you to insert only the ad that is related to the niche of that particular post and thus will increase your blog profit day after day. Ad rotate is also possible with the use of shortcode to create ad group which will display different advertisement at different day on that particular post. 
   
== Frequently Asked Questions == 

= How to open the 'Smart AD Tags' setting page? =
 
Click on 'Smart Ad' tab on the left sidebar of your wordpress admin panel to open the setting page.

= How to do the set up and how many features do this plugin has? = 

Read below article and find out more about how to use this plugin and the features it provides

== Installation ==

1) Upload Smart AD Tags plugin to your blog, then activate it!
2) Then follow the instructions under FAQ to do the settings.
